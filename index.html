Welcome to myopenHAB!

myopenHAB is an instance of the openHAB Cloud service, which is hosted by the openHAB Foundation e.V.

The openHAB Cloud service is useful for different use cases:

  • It allows remote access to local openHAB instances without having to expose ports to the Internet or to require a complex VPN setup.
  • It serves as a connector to Google Cloud Messaging (GCM) and Apple Push Notifications (APN) for pushing notifications to mobile phone apps.
  • It brings integration possibilities with services that require an OAuth2 authentication against a web server, such as IFTTT or Amazon Alexa Skills.

We are offering this service completely for free. It is meant to allow users to quickly check out its features without having to set up and host a personal instance. Although it is mainly meant as a demonstrator, it is absolutely fine to use it for your production system as well. You must be aware though that we cannot offer any SLAs regarding availability etc, but we promise to keep it up and running to the best of our capabilities. Please read our Terms of Use for all details.

In order to use myopenHAB, you will need to install the openHAB Cloud Connector bundle. Please check the documentation for openHAB 1.8 or for openHAB 2.0.

If you want to use myopenHAB through your native Android and iOS apps, please enter "https://myopenhab.org" as a remote url and your myopenHAB account username and password as credentials.
